
# Face Detection

face detection app using opencv


## Run Locally



Clone the project

```bash
  git clone https://github.com/superiorkid/face-detection.git
```

Go to the project directory

```bash
  cd face-detection
```

Install package required

```bash
  pip install -r requirements.txt
```

Start

```bash
  python facedetection.py
```
